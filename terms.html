OpenFlat 自动续费协议

欢迎您使用 OpenFlat 自动续费服务。

当您点击“订阅”“开通自动续费”或实际使用自动续费功能时，即视为您已阅读并同意本协议全部内容。

如您在计费周期到期前未主动取消自动续费，我们将按约定在到期日自动扣除下一周期费用并延长服务有效期。

一、协议说明

本《OpenFlat 自动续费协议》（以下简称“本协议”）是《OpenFlat 服务协议》《OpenFlat 服务购买须知》（https://openflat.tech/rules.html）的补充协议，与其构成统一整体。

如本协议与上述协议或购买须知存在不一致之处，以本协议约定为准。

本协议适用于您通过 OpenFlat 官方网站、OpenFlat 客户端、Apple App Store 或其他官方渠道开通的自动续费服务。

二、自动续费服务说明

自动续费服务是指：在您已开通 OpenFlat 付费服务的前提下，如您选择自动续费功能，则在当前计费周期届满前，系统将根据您选择的计费周期自动扣除下一周期费用，以延续您的服务有效期。

计费周期

计费周期可能包括：

  • 月度
  • 季度
  • 年度

（具体以您购买页面显示为准）

自动续费生效规则

如您在当前计费周期到期前未主动取消自动续费，则视为您授权 OpenFlat 在到期日自动发起下一周期扣费。

三、自动扣款规则

3.1 支付渠道说明

自动续费扣款方式取决于您开通时所选择的支付渠道：

（1）微信支付 / 支付宝 / 银行卡等第三方支付方式

  • 到期日当天系统将自动发起扣款；
  • 若因余额不足、限额、账户异常等原因扣款失败，我们可能会在短期内发起重试；
  • 若多次重试仍失败，则自动续费服务将终止，您的服务将在当前周期结束后失效。

（2）苹果应用内支付（Apple IAP）

  • 若您通过 Apple ID 订阅，扣款规则以 Apple 官方订阅规则为准；
  • Apple 将在订阅期到期日自动扣费；
  • 若扣款失败，将进入 Apple 官方重试期；
  • 重试期结束仍未成功扣费的，订阅自动终止。

Apple 订阅取消路径：iOS 设备 → 设置 → Apple ID → 订阅 → 选择 OpenFlat → 取消订阅

3.2 扣款授权

您确认并同意：

  • OpenFlat 可在无需再次验证您的账户密码、支付密码或短信验证码的情况下，按本协议约定发起自动扣款；
  • 扣款行为视为您本人操作。

3.3 账户风险说明

您应自行确保：

  • 账户状态正常；
  • 余额充足；
  • 支付方式未过期；

因账户异常、余额不足、支付限额等导致续费失败的，由您自行承担后果。

3.4 第三方条款

如您通过第三方支付渠道完成支付，您还需同时遵守该第三方平台的服务条款及支付规则。

四、取消自动续费

您可随时取消自动续费服务。

4.1 OpenFlat 官方渠道取消方式

您可通过以下路径取消：

  • 登录 OpenFlat 官网
  • 进入「账户中心」
  • 点击「订阅管理」
  • 选择「自动续费管理」
  • 点击「取消自动续费」

取消成功后，自动续费将在当前计费周期结束后停止。

4.2 Apple 订阅取消

如您通过 Apple ID 订阅，需在 Apple 系统内取消：

iOS 设备 → 设置 → Apple ID → 订阅 → OpenFlat → 取消订阅

4.3 取消生效说明

  • 自动续费状态更新可能存在延迟，以 OpenFlat 系统显示为准。
  • 在您成功取消前已发起的扣款指令仍然有效。
  • 已扣除的费用原则上不予退还（法律另有规定除外）。

五、续费提醒通知

为保障您的知情权，OpenFlat 可能会在自动扣款前通过以下方式通知您：

  • 站内通知
  • 邮件
  • 短信
  • 应用内提醒

具体提醒方式以实际可用方式为准。

六、价格调整

OpenFlat 有权根据运营情况调整服务价格。

若价格调整，我们将通过公告、邮件或站内通知方式提前告知。

若您不同意价格调整，可在到期前取消自动续费。

如未取消，则视为同意按调整后的价格续费。

七、服务终止

OpenFlat 有权基于运营调整、技术原因或合规要求停止自动续费服务。

届时我们将提前通知，并在当前周期结束后停止服务。

八、协议有效期

本自动续费协议自您开通之日起生效，长期有效，直至：

  • 您主动取消；
  • 因违反平台规则被取消；
  • 或 OpenFlat 停止该服务。

九、违约处理

如您违反：

  • OpenFlat 服务协议
  • 服务购买须知
  • 平台使用规则

OpenFlat 有权暂停或终止您的自动续费服务，并保留追究法律责任的权利。

十、隐私保护

您使用自动续费服务所涉及的个人信息处理，适用《OpenFlat 隐私政策》。
